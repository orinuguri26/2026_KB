1. link 태그
   - rel
     현재 문서와 연결된 리소스의 관계(relationship) 말함
     해당 링크가 어떤 용도로 쓰이는지 나타냄
   - href
     어디로 연결할지 주소를 지정(hypertext reference)

2. script 태그
   - src로 연결할 JS파일 잡아옴
   - JS 내에선 scripts태그 없이 작성

3. emmet약어
   - 태그명 \* 수, 수 삽입은 $로, 내용은 {}로 묵어서

4. span
   - rowspan
     행을 합친 크기 만큼 나옴
   - colspan
     열을 합친 크기만큼 나옴

5. img
   body태그안에서 뿌림
   - alt
     해당 그림이 없을 때 나올 대체 문구
   - width, height
     그림 사이즈 지정 가능
6. audio, video
   - controls="controls"
     재생 도구를 출력
