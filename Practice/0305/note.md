1. label태그
   form태그의 요소에 이름(설명)을 붙여주는 태그
   for=""로 해당 태그의 id와 연결, 아니면 해당 input태그를 감싸면됨

2. input의 type인 radio
   name으로 같은 선택지들에게 동일한 값을 줘야 묶임

3. select 태그
   - 드롭다운 만들 수 있음
   - option태그로 안에 선택지 생성
   - select 태그에 multiple속성을 통해 다중 선택도 가능

4. optGroup
   - option들을 그룹화 하여 드롭다운 함
   - optGroup태그의 속성으로 label을 넣으면 해당 그룹의 이름 지정가능

5. p태그와 span 태그
   - p태그는 블록요소라서 줄바꿈이 있고 한줄 차지
   - span태그는 인라인 요소로 줄바꿈 없고 여백도 없음
