봇(bot), 크롤러 라 한다. 
사용자가 검색했을때

검색엔진 최적화란? (SEO) : 검색이 잘 되도록 설계하는 일
HTML로 웹페이지를 만들면서 검색 엔진 최적화하는 방법 2가지 
1. 메타 태그 : 검색 엔진에 정보를 제공할 목적인 태그
2. 시맨틱 태그: 웹 페이지 구조를 구성하는 태그 

-레이아웃용 시맨틱 태그
 <header> : 소개 요소 또는 탐색 링크 
 <nav> : 탐색 링크 정의한다. 주로 메뉴
 <section> : 본문 내용을 포함하는 공간  
 <article> : 독립적인 요소 
 <aside> : 사이드바처럼 좌,우 위치하는 공간 
 <footer> : 문서의 하단을 정의 , 보통 카테고리 요약하거나 저작권 명시
