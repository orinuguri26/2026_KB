   기능구현 
   - 계산기 UI에서 숫자(Operand), 사칙연산(operator) 입력 
   - 입력된 데이터를 출력 
   => 입력값을 저장하는 데이터를 정의 
      필요한 부분을 바인딩 처리 해야한다. 

   1. App.vue 
     1-1 데이터(변수) 정의 
      - 출력데이터 저장 => output
      - 이전에 입력된 숫자나 연산결과를 저장 => prev
      - 현재 입력중인 숫자를 저장  => cur
      - 사용자가 입력한 연산자를 저장  =>operator

     1-2 데이터중 HTML요소와 바인딩 데이터는? 
         사용자에게 계속 노출되는 데이터 => output  => 양방향 바인딩 처리 
 

     1-3 이벤트 연결      
        계산기 UI에 필요한 이벤트를 연결 
        output 버튼을 제외한 숫자10개 , . , (+,-,*,/),C 버튼이 이벤트 연결대상이다.

        클릭이벤트 처리(@click) 연결한다. 
        클릭 이벤트에 따른 이벤트 핸들러 작성  => operation() 

     2. 계산기 로직(logic : 프로그램이 수행해야 하는 작업 또는 계산순서와 조건을 정의과정) 구현     

        계산 담당하는 로직은 클릭 이벤트에 연결한 operation() 내부에 작업한다. 

       2-1 클릭한 버튼 값을 가져오기 
         <input> value 속성에 할당된 값을 가져온다. 
         input 속성의 값을 이벤트 객체가 가져와서 출력 하는 코드 작성    

       2-2. 입력값  e.currentTarget.value  => output 
            이벤트가 발생한 UI 값을 출력칸에 노출되도록 로직 
            <input v-model="output" type="text" name="output" readonly/> 

            1. 버튼을 클릭하면 해당 숫자를 n 변수에 저장 
               이때 저장된 숫자는 내부적으로 문자열데이터이다. 숫자처럼 보이지만 실제로는 문자열이다.

            2. 사용자가 입력한 숫자를 cur 데이터에 저장한다. 
               자바스크립트에서는 문자열끼리 덧셈하면  문자열이 하나로 연결된다.  "1+1"   "1"+ "1+" "1+1" 
               this.cur += n;
               cut 데이터를 output에 저장한다. 
               현재 cur에 저장된 문자열을 output 에 출력한다.
