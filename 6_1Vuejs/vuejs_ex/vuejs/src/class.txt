1. 컴포넌트와 컴포넌트의 활용법
   컴포넌트 : 어플리케이션을 구성하는 재사용 가능한 요소
   고객요청서1 : 앞바퀴 휠과 타이어를 새로 장착해 주세요 (타이어 노랑)
   고객요청서2 : 뒷바퀴 휠과 타이어를 새로 장착해 주세요
   자동차 본체 노란색

   자동차 작업 순서
   앞바퀴 휠 장착 => 뒷바퀴 휠 장착 => 타이어장착 => 타이어 커스터마이징 (고객의 기호에 맞게 제품의 일부를 변경하는 행위)

   App.vue  => 모든 컴포넌트의 (부모 ,최상위 컴포넌트) => root

   문제 : root가 언제 임포트되어 사용되었을까?
   답   : vue 어플리케이션에서 처음 실행되는 src/main.js 가 실행되었을때
   vue의 실행 순서
   1. 웹 브라우저에서 index.html 로드
   2. 로드하는 중 , src/main.js 파일을 로드하고 실행
   3. 실행 중인 root인 App.vue를 임포트하고 사용


   1. 기본 템블릿 문법
      1-1 텍스트 출력
          Vue에서 텍스트를 동적으로 출력하는 대표방법 => {{}} 머스태시 문법
          v-text 디렉티브를 이용하여 동적 출력
       디렉티브(directive)
       템블릿 블록에서 HTML요소에 입력하는 "v-"시작하는 특수한 속성
       동적으로 텍스트를 출력하거나 클래스를 표시, 조건에 따라 일부 요소 출력 제어
       반복, 이벤트를 다루는 역할

       JSON(JavaScript Object Notation)
       가볍고 직관적인 문자열 기반의 데이터 형식
       데이터 저장과 교환에 사용된다.
       자바스크립트 애플리케이션에서 자바 어플리케이션으로 배열 데이터를 전달할때
       두언어의 배열처리 방식이 달라 직접적으로 호환이 되지 않는다.
       배열데이터를 JSON으로 변환하면 자바에서도 사용할 수 있다.


      - 반응형 상태
      - 컴퓨티드
      - 데이터 바인딩
