레이아웃  => 이미지,텍스트 등을 원하는 위치로 배치하는 것 

1.플렉스(유연한)박스란?
  사용자인터페이스(UI) 디자인에 최적화된 레이아웃을 정의하는 CSS
2.플렉스박스가 필요한 순간은?
3.플렉스박스로 정렬하기
4.플렉스박스로 축 방향 바꾸기 

플렉스박스 제 1원칙 : 부모만 플렉스 할 수 있다. 
       부모자식관계를 파악한 후 부모에 적용해야 자식이 영향을 받는다.

       사용방법:   
            <style> 
                body {
                    display : flex; 
                }
            </style>

            1. 공간에 맞추기 : display flex (자식요소를 원하는 방향으로 유연하게 배치할 수 있는 속성)
               display 속성은 block, inline, flow, grid, flex 
            
            2. 주축 정렬 : justify-content 속성 
               웹페이지의 축을 따라 요소 사이에 공간을 만드는 방법을 정의한다. 
               축 => 주축(가로), 교차축 (세로)  
               flex-direction 속성을 사용하여 축을 변경이 가능하다. 

               플렉스박스는 두 축을 기준으로 움직이기 때문에, 축방향을 기준으로 요소를 움직인다.

            3. 교차(세로)축 정렬 
               align-items 속성 

            4. 플렉스 축 방향 바꾸기 : flex-direction 속성 

                body {
                    flex-direction : column; (세로방향), row(기본방향,가로) , row-reverse (역순으로 배치)
                }
