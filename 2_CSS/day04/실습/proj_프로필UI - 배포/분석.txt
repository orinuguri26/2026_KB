1. rem => 글꼴의 크기를 지정하는 단위 
          상대단위로 설정하면 크기가 해상도에 따라 유동적으로 변한다. 
          html 요소의 기본값이 16px이므로 기본 상태에서
         1rem = 16px 이다. 
         html의 요소의 값이 바뀌면 상대적으로 조정된다.
   px => 절대 단위이다. 즉 페이지 크기를 줄이든 키우든 같은 크기를 가진다.


2. overflow속성  => 내용이 요소 크기를 벗어났을때 어떻게 처리할 지를 정하는 속성 
                 CSS는 기본값으로 visible 속성을 가지고 있다. 
                 상자크기와 상관없이 요소를 보여준다. 
                 hidden으로 바꾸면 넘치는 컨텐츠는 보이지 않는다.
